# Investigation Skill

## Ethical Framework

- **Public information only** — No hacking, no unauthorized access
- **Verify before reporting** — Cross-reference across sources
- **Respect privacy** — Do not aggregate data to enable harassment
- **Legal compliance** — Follow applicable laws for the jurisdiction
- **Proportionality** — Match investigation depth to legitimate need

## Methodology

### Phase 1: Scoping

1. Define the subject (person, company, domain, event)
2. Clarify the purpose (due diligence, hiring, business decision)
3. Set boundaries (what sources are appropriate)
4. Confirm authorization from the requesting party

### Phase 2: Collection

#### People

| Source | What to Find |
|--------|-------------|
| LinkedIn | Role history, connections, endorsements |
| Public records | Court filings, property, business registrations |
| News/Media | Mentions, interviews, articles |
| Academic | Publications, patents, presentations |
| Social media | Public posts, interests, affiliations |
| Corporate filings | Board positions, SEC filings, beneficial ownership |

#### Companies

| Source | What to Find |
|--------|-------------|
| Secretary of State | Registration, officers, status |
| SEC/EDGAR | Filings, financials, insider trades |
| News | Press coverage, controversies, acquisitions |
| Court records | Lawsuits, regulatory actions |
| Reviews | Glassdoor, BBB, customer reviews |
| Tech | Stack, domains, patents, open source activity |

#### Domains/Infrastructure

| Source | What to Find |
|--------|-------------|
| WHOIS | Registrant, dates, registrar |
| DNS | Records, mail servers, subdomains |
| Certificate transparency | SSL certs, related domains |
| Web archive | Historical snapshots, changes |
| Technology | Framework detection, hosting provider |

### Phase 3: Analysis

1. **Cross-reference** findings across sources
2. **Timeline** key events chronologically
3. **Identify patterns** — connections, anomalies, gaps
4. **Assess credibility** of each source
5. **Flag uncertainties** — what couldn't be confirmed

### Phase 4: Reporting

```markdown
## Investigation Report: [Subject]

**Date**: [Date]
**Scope**: [What was investigated and why]
**Confidence**: [High/Medium/Low]

### Summary
[2-3 paragraph executive summary]

### Key Findings
1. [Finding — source — confidence level]
2. [Finding — source — confidence level]

### Timeline
- [Date] — [Event]
- [Date] — [Event]

### Red Flags
- [Concern — why it matters]

### Gaps
- [What couldn't be determined and why]

### Sources
- [Source with URL/reference]
```

## Rules

- Always state the confidence level for findings
- Never present speculation as fact
- Cite specific sources for every claim
- Clearly separate confirmed facts from inferences
- If asked to investigate something unethical, decline and explain why
