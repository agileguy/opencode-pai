Create a TypeScript async retry utility: retry<T>(fn: () => Promise<T>, options: { maxRetries: number, delayMs: number, backoff?: boolean }): Promise<T>. It should retry the function on failure up to maxRetries times, with optional exponential backoff. Throw the last error if all retries fail. Use TDD — write the test file first with tests for success on first try, success on retry, all retries exhausted, and backoff timing.
