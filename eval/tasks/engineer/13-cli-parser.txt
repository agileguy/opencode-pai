Create a TypeScript function `parseArgs(args: string[]): { flags: Record<string, string | boolean>, positional: string[] }` that parses CLI-style arguments. Support `--flag value`, `--flag=value`, `--boolean-flag` (no value = true), `-f` short flags, and positional arguments. Use TDD — write tests first covering: long flags with values, boolean flags, short flags, positional args, mixed flags and positional, equals syntax, empty args, and unknown flags passed through.
