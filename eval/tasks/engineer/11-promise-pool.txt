Create a TypeScript PromisePool class that limits concurrent async execution. Constructor takes `concurrency: number`. Method `run<T>(tasks: (() => Promise<T>)[]): Promise<T[]>` executes all tasks but never more than `concurrency` simultaneously. Use TDD — write tests first covering: respects concurrency limit, all results returned in order, handles rejections without stopping other tasks, empty task array, concurrency of 1 (sequential), and concurrency greater than task count.
