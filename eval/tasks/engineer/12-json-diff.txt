Create a TypeScript function `deepDiff(a: unknown, b: unknown): Diff[]` that compares two JSON-compatible values and returns an array of differences. Each Diff has `{ path: string, type: 'added' | 'removed' | 'changed', oldValue?: unknown, newValue?: unknown }`. Use TDD — write tests first covering: identical objects, added keys, removed keys, changed values, nested objects, array differences, null vs undefined, and empty objects.
